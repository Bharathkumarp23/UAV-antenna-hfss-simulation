# UAV-antenna-hfss-simulation
Coaxial-fed microstrip patch antenna (27 ×27 mm) on Rogers RO4232 substrate, designed for UAV communication. Simulated in ANSYS HFSS with parameterized geometry, PEC ground plane, and driven terminal solution. Analyzes S11, gain, radiation pattern, and impedance bandwidth for UAV  applications.


## 🛠️ Tools Used
- ANSYS HFSS (High Frequency Structure Simulator)

## 📁 Files Included
- `uavantfr.hfss` - Main HFSS simulation project file
